# Calendario

La vista de Calendario muestra las tareas en una cuadrícula visual según sus fechas de vencimiento.

## Vistas

- **Semana** — muestra las tareas distribuidas en los 7 días de la semana actual
- **Mes** — muestra las tareas como puntos en un calendario mensual

## Navegación

- Usa las flechas **Anterior / Siguiente** para moverte entre semanas o meses
- Haz clic en **Hoy** para volver al período actual

## Filtros

El Calendario soporta los mismos filtros que la vista de Tareas:
- Área
- Dispositivo
- Prioridad

## Puntos de tarea

Las tareas aparecen como puntos de color en su fecha de vencimiento. El color del punto corresponde al nivel de prioridad de la tarea:

| Prioridad | Color del punto |
|---|---|
| Baja | Verde |
| Media | Amarillo |
| Alta | Naranja |
| Crítica | Rojo |

Haz clic en un punto o en un día para ver las tareas que vencen en esa fecha.
