## Commands

```bash
npm start          # Dev server at localhost:3000 (hot reload)
npm run build      # Production build into build/
npm run serve      # Serve the production build locally
npm run typecheck  # TypeScript type check (no emit)
npm run clear      # Clear Docusaurus cache (.docusaurus/ and build/)
```

## Architecture

This is a [Docusaurus 3](https://docusaurus.io) documentation site for the IntelliKeep Home Assistant integration, written in TypeScript with the classic preset.

**Key files:**
- `docusaurus.config.ts` — site config: URL, navbar, footer, plugins, theme
- `sidebars.ts` — sidebar is fully autogenerated from `docs/` folder structure
- `src/pages/index.tsx` — marketing homepage (separate from the docs)
- `src/css/custom.css` — global CSS overrides and theme variable overrides

**Docs structure (`docs/`):**
Sidebar order and nesting are controlled by `sidebar_position` frontmatter and directory hierarchy. The sidebar is autogenerated — no manual sidebar entries needed.

**Conventions:**
- All navigation items (navbar labels, sidebar category labels, page titles) must start with an uppercase letter.
- Sidebar category labels for subdirectories are set via `_category_.json` files (`label` field).

**Logo:**
- Light mode: `static/img/icon.png`
- Dark mode: `static/img/logo-dark.png`
Both are sourced from the IntelliKeep brand assets at `../intellikeep/brand/`.

**Production URL:** `https://intellikeep.intellilar.com`
